
📌 توضیح کدها
کد	توضیح
کد شبیه‌سازی‌شده	یک برنامه‌ی کامل با PySide6 که فایل‌ها را بر اساس پسوند و کلمات کلیدی در نام فایل دسته‌بندی می‌کند. این کد بدون نیاز به هوش مصنوعی کار می‌کند و برای تست اولیه مناسب است.
کد اتصال به Gemma	یک کلاس برای ارتباط با مدل Gemma از طریق API Ollama. شامل متدهای chat (مکالمه) و analyze_image (تحلیل تصویر) است.
🚀 نحوه‌ی استفاده
۱. اجرای کد شبیه‌سازی‌شده:
کد را در فایل main.py ذخیره کنید.

کتابخانه‌های مورد نیاز را نصب کنید:

cmd
pip install pyside6
برنامه را اجرا کنید:

cmd
python main.py
۲. استفاده از کد اتصال به Gemma:
مطمئن شوید که Ollama در حال اجرا است (ollama serve).

مدل Gemma را در Ollama دانلود کنید (ollama pull gemma3:4b).

کلاس GemmaConnector را در برنامه‌ی خود استفاده کنید.
