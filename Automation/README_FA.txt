راهنمای AutoSync برای SmartArchive

این ابزار هر 60 ثانیه تغییرات ذخیره‌شده در پروژه را بررسی می‌کند. اگر تغییری وجود داشته باشد، به صورت خودکار:
1) git add -A
2) git commit
3) git push origin main
را اجرا می‌کند.

نصب:
1) پوشه Automation را بدون تغییر داخل این مسیر قرار دهید:
   C:\Users\amin\Desktop\SmartArchive\Automation
2) روی Install_AutoSync.bat دابل‌کلیک کنید.
3) پنجره باید پیام "AutoSync is installed and running." نشان دهد.

لاگ:
C:\Users\amin\AppData\Local\SmartArchive\AutoSync.log

توقف کامل:
روی Remove_AutoSync.bat دابل‌کلیک کنید.

نکات مهم:
- این ابزار فقط تغییراتی را ارسال می‌کند که واقعاً Save شده‌اند.
- فایل‌های نادیده‌گرفته‌شده در .gitignore ارسال نمی‌شوند.
- اگر اینترنت قطع باشد یا Push خطا دهد، پیام خطا داخل فایل Log ثبت می‌شود و در اجرای بعد دوباره تلاش می‌کند.
- این ابزار Pull یا Merge خودکار انجام نمی‌دهد. اگر شخص دیگری از کامپیوتر/حساب دیگری روی GitHub تغییر داده باشد، ابتدا باید آن تغییر را با GitHub Desktop دریافت و بررسی کنید.
- چون همه روی یک کامپیوتر کار می‌کنید، فقط یک نفر در هر لحظه روی فایل‌ها تغییر ایجاد کند.
