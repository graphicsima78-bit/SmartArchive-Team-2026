به‌روزرسانی SmartArchive - نسخه 9

این نسخه شامل تمام تغییرات v8 و این اصلاح نام فایل صوتی است:

- اگر Tag داخلی آهنگ، خواننده و آلبوم وجود داشته باشد:
  موسیقی\خواننده\آلبوم\نام آهنگ - خواننده.ext

- اگر Tag اطلاعات کافی نداشته باشد، برنامه حدس اشتباه نمی‌زند.
  فقط صفرهای ابتدای نام فایل را برمی‌دارد و بقیهٔ نام را حفظ می‌کند.

مثال:
000000000Reza ramy.mp3
به:
Reza ramy.mp3

- فایل اصلی مبدأ تغییر نام نمی‌گیرد؛ فقط نسخهٔ دسته‌بندی‌شده در مقصد نام تمیزتری می‌گیرد.

روش اعمال:
1) ArchivePro را ببندید.
2) روی Apply_SmartArchive_Update_v9.bat دابل‌کلیک کنید.
3) در هشدار Windows فقط اگر نام فایل درست است روی Run بزنید.
4) AutoSync تغییرات را خودکار Commit و Push می‌کند.
