به‌روزرسانی SmartArchive - نسخه 16

این نسخه سیستم کنترل‌شدهٔ دسته‌بندی هوشمند و Tab پروژه‌ها را اضافه می‌کند.

مدیریت دسته‌بندی‌ها:
- Tab جدید «مدیریت دسته‌بندی‌ها».
- فقط مسیرهای مجاز یا مسیرهای اضافه‌شده توسط کاربر ساخته می‌شوند.
- مثال: sandwich → 01_تصاویر\غذا\فست‌فود\ساندویچ.
- پوشه‌های نامشخص به دسته‌بندی نشده یا نیازمند بررسی می‌روند، نه پوشه‌های تصادفی.
- می‌توانید مسیر فارسی و نام‌های مشابه را در Tab جدید اضافه کنید.
- فایل taxonomy.json شامل قوانین شماست و در به‌روزرسانی‌های بعدی حذف یا جایگزین نمی‌شود.

پروژه‌ها:
- Tab جدید «پروژه‌ها».
- حالت معماری و سه‌بعدی: همه فایل‌ها زیر 12_پروژه‌ها\01_معماری_و_سه‌بعدی\سال\نام پروژه می‌روند.
- حالت تولید محتوا: همه فایل‌ها زیر 12_پروژه‌ها\02_تولید_محتوا\پلتفرم\نوع محتوا\نام پروژه می‌روند.
- فایل‌های معماری، مدل، رندر، خام، صوت، گرافیک، متن و خروجی نهایی در پروژه کنار هم می‌مانند.

روش اعمال:
1) ArchivePro را ببندید.
2) روی Apply_SmartArchive_Update_v16.bat دابل‌کلیک کنید.
3) در هشدار Windows فقط اگر نام فایل درست است روی Run بزنید.
4) AutoSync تغییرات را خودکار Commit و Push می‌کند.
