به‌روزرسانی SmartArchive - نسخه 7

این نسخه شامل همهٔ قابلیت‌های v6 و بهبود نام‌گذاری موسیقی است.

نام‌گذاری موسیقی:
- اگر متادیتای داخلی فایل وجود داشته باشد، نسخهٔ آرشیوی با این فرمت نام‌گذاری می‌شود:
  نام آهنگ - خواننده.mp3
- مثال:
  Hello - Adele.mp3
- اگر Artist و Album در Tag فایل وجود داشته باشد، مسیر مقصد می‌شود:
  موسیقی\Artist\Album\نام آهنگ - خواننده.ext
- فقط نسخهٔ دسته‌بندی‌شده تغییر نام می‌گیرد؛ فایل اصلی دست‌نخورده می‌ماند، مگر کاربر حالت انتقال و حذف مبدأ را فعال کند.
- اگر Tag آهنگ/خواننده/آلبوم ناقص باشد، فایل اصلی با نام فعلی حفظ می‌شود و حدس نادرست زده نمی‌شود.

روش اعمال:
1) ArchivePro را ببندید.
2) روی Apply_SmartArchive_Update_v7.bat دابل‌کلیک کنید.
3) در هشدار امنیتی، فقط اگر نام فایل درست است روی Run بزنید.
4) AutoSync تغییرات را خودکار Commit و Push می‌کند.
