به‌روزرسانی SmartArchive - نسخه 4

این به‌روزرسانی شامل:
- دسته‌بندی کامل‌تر فایل‌ها بر اساس پسوند، از جمله MAX، SKP، RVT، STL، 3DS، Office، فایل‌های فشرده، نرم‌افزار، پایگاه‌داده و بازی
- انتقال فایل MAX به سه‌بعدی/3ds_Max
- انتقال لوگو به تصاویر/اشیاء/لوگو
- گزینه حذف فایل مبدأ پس از کپی موفق (به‌صورت پیش‌فرض خاموش)
- گزینه پردازش دوباره فایل‌های ثبت‌شده، برای تست مجدد با تنظیم تاریخ متفاوت
- نمایش علت دسته‌بندی تصویر در Log

روش اعمال:
1) ArchivePro را ببندید.
2) فایل Apply_SmartArchive_Update_v4.bat را دابل‌کلیک کنید.
3) در هشدار Windows فقط اگر نام فایل همین Apply_SmartArchive_Update_v4.bat است، Run را بزنید.
4) منتظر پیام موفقیت بمانید.
5) VS Code تغییر فایل‌ها را تشخیص می‌دهد و AutoSync آن‌ها را خودکار Commit و Push می‌کند.
